Built for Canadian regulations

Cumulo Flight Deck calculates night time per RAC 101.01 (sunrise/sunset, not approximations). Cross-country detection per CAR 401.34 (great-circle distance > 25 NM). The PDF export matches TP 14052 column-for-column for ramp checks and ATPL submissions. Multi-airline import — Navblue iCal, CrewTrac PDF (Jazz), Sabre PDF (WestJet), AIMS (Air Canada), plus 5 CSV importers (ForeFlight, LogTen Pro, MyFlightbook, Logbook Pro, Safelog).
