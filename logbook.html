Paste your Navblue calendar subscription URL. Cumulo will fetch your roster automatically — completed flights are imported with real block hours, future flights are skipped.

👨‍✈️ Crew names from Navblue PDF roster

The Navblue iCal feed doesn't include captain names. Drop your HrRosterReport PDF here (download it from Navblue when convenient — once per month is enough). Cumulo extracts the captain name from each flight and merges it into your logbook automatically. Nothing is uploaded — parsing happens 100% in your browser.

Export Logbook as PDF
Landscape print-ready logbook. CAR 401.08 column layout with running totals and signature block. Use this for Transport Canada inspections, employer submissions, or just to print on paper.
📊
Visible columns (Logbook table + PDF export)
All data is captured on every flight (Transport Canada CAR 401.08 + Standard 421 categories). Pick which columns you want visible in the table and PDF. Defaults : essential 705 F/O view. Add more for ATPL preparation, recurrent training, or detailed audit.
🌙
Recalculate Night & Cross-Country
Re-process all imported flights to compute night hours (RAC 101.01) and cross-country (CAR 401.34 — > 25 NM) using astronomical sunrise/sunset and great-circle distance. Run this once after importing from Navblue iCal — older imports had no day/night split.
↩️
Undo last bulk operation
Cumulo automatically snapshots your logbook before any bulk operation (sync, recalculate, etc.). If something went wrong, restore the previous state with one click. Your flight data is precious — Cumulo never deletes without explicit per-flight confirmation.
💾
Download Data Backup (JSON)
Technical backup of all your flight data in JSON format. Use this only to restore Cumulo on another device or recover from a bug. This file is not human-readable — use Export PDF instead if you want to read your logbook.
