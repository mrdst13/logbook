👤 Pilot Information
Comma-separated airline IATA codes. Cumulo recognizes flights matching these prefixes when syncing iCal / PDF rosters. The dropdown above auto-fills this for you.
Under 705 ops, virtually every flight terminates with an instrument approach (ILS / RNAV / VOR / visual after IAP). When this is on, Navblue sync sets Approaches = 1 on each new flight imported, feeding the CAR 401.05 currency counter (6 approaches / 6 months). Auto-enabled when you select a 705 airline above. Turn off for 704 / 703 / bush ops where flights are often VFR — enter approaches manually per flight. Existing flights and your manual entries are never overwritten.
Default: OFF (anonymized). When OFF, Cumulo automatically stores captain names from Navblue / PDF roster / AI photo imports as initials only (e.g. M.D. instead of Martin Daoust). This protects third-party privacy under PIPEDA Principle 4.3 (consent from the data subject).

Turn ON only if you've obtained your crew's consent to record their names in your logbook, or if you take responsibility for that handling. Already-imported names are not touched retroactively — only new imports follow the current setting.
Cleaner logbook view — any column where every flight shows 0 is hidden from the table on screen. Useful when you've imported a CSV that doesn't have the full TC granularity (e.g. ForeFlight imports leave SE Day, IFR Hood, etc. empty). The PDF export for Transport Canada always keeps all 38 columns for ramp-check compliance, regardless of this setting.
Check the configurations you fly. Used to filter aircraft type suggestions and currency tracking (TC requires separate currency for floats, skis, tailwheel per CAR 401.05).
